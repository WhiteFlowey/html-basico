Estrutura Básica do HTML

<html></html> — Tem o objetivo de marcar e formatar o conteúdo da página web informando ao navegador como exibir e interpretar a informação.

<head></head> — Serve como cabeçalho de um documento contendo metadados (informações sobre a página) que não são exibidos visualmente na tela, mas são essenciais para navegadores e motores de busca.

<body></body> — Contém e exibe todo o conteúdo vísivel de uma página HTML como texto, imagens, links e outros elementos que o usuário interagirá.

<title></title> — Define o título de uma página web, que aparece na barra de abas do navegador e nos resultados de pesquisa

<hr> — Serve para inserir uma linha horizontal que representa uma quebra temática ou uma divisão de conteúdo em uma página web, como uma mudança de tópico ou cena.

<small></small> — Diminui o tamanho da fonte de um texto.

Falando sobre Tags

<i></i> — Exibe o texto em itálico.

<strong></strong> — Indica que o texto contido nela tem grande importância, ênfase ou seriedade.

<input> — Cria campos de entrada de dados em um formulário ou página web.

<img> — Exibe imagens numa página HTML.

Atributos das Tags

Atributos palavras-chave adicionadas a uma tag de abertura para fornecer informações adicionais ou controlar o comportamento de um elemento, funcionando como modificadores que definem a função ou a aparência de um elemento.

<input> — Possui atributos para identificar a entrada com exemplo <input type="text"> que disponibiliza um campo para escrever texto.

<img> — Exemplo de uso <img src="arquivo/caminho">

id="titulo" — Define um identificador único para um elemento e serve para aceder, estilizar ou referenciar esse elemento específico no documento. Consegue chamar usando o símbolo de # em CSS e para criar links de âncora para partes específicas de uma página.

style="color: blue" — Aplica um estilo diretamente em um elemento HTML.

class="titulo" — Agrupa elementos que partilham características semelhantes e aplicar estilos ou manipulações em grupo através de CSS ou JavaScript. Ao atribuir a mesma classe a múltiplos elementos, você pode definir um estilo para essa classe uma vez e fazê-lo ser aplicado a todos os elementos que a possuem, tornando o código mais eficiente e reutilizável.

width — Largura.

Textos

<h1></h1> — Define o título principal de uma página da web, destacando o tema central do conteúdo para os usuários e os motores de busca e varia entre h1-h6 quanto maior o número menor o tamanho da letra.

<p></p> — Define um parágrafo de texto.

<blockquote></blockquote> — Marca e destaca citações longas de outra fonte, isolando-as do restante do conteúdo da página.

<u></u> — Sublinha um texto

<mark></mark> — Destacar e marcar trechos de texto dentro de um documento, similar a um marca-texto físico.

<sup></sup> — Exibe texto como sobrescrito, ou seja, texto menor e alinhado na parte superior do texto principal. Ela é usada para formatos como expoentes (ex: x²).

Listas Ordenadas e não ordenadas

<ol></ol> — Cria uma lista ordenada, ou seja, uma lista onde a ordem dos itens é importante e é exibida com marcadores como números.

<ul></ul> — Cria uma lista não ordenada, ou seja, uma lista de itens que não tem uma ordem específica, sendo apresentada com marcadores.

<li></li> — serve para definir cada item individual dentro de uma lista em HTML, sendo utilizada tanto em listas ordenadas quanto em listas não ordenadas para agrupar conteúdos relacionados em uma estrutura de lista.

Links

<a></a> — Cria hiperlinks que direcionam o usuário para outra página web, um recurso específico na mesma página ou para outros recursos como imagens e arquivos. <a href="link" target="_blank/_self"></a> — _blank abre uma nova guia,_self fica na mesma guia.

<a href="#nome_id">link</a> — Cria um link que leva ao id.
